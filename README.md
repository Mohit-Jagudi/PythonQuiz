# PythonQuiz 

A simple and interactive **Python-based Quiz Game** that tests your knowledge across multiple subjects like **Python, C, C++, DBMS, and Cyber Security**.

---

##  Features

*  Multiple Subjects:

  * Python
  * C
  * C++
  * DBMS
  * Cyber Security

*  Mix Mode (Random questions from all subjects)

*  10-second timer for each question

*  Leaderboard system (scores saved in file)

*  Input validation (no crash on wrong input)

*  Beginner-friendly CLI interface

---

##  Technologies Used

* Python
* File Handling
* Random Module
* Time Module

##  How to Run

1. Download the project files (ZIP or individual files)

2. Create a new folder on your system
   Example: `PythonQuiz`

3. Move all project files into that folder:

   * `quiz.py`
   * `questions.py`

4. Open this folder in any code editor (like VS Code)

5. Open terminal inside the editor

6. Run the program using:

```bash
python quiz.py
```

7. Follow the instructions on screen to play the quiz 

---

##  How to Play

1. Enter your name
2. Choose subject (or MIX mode)
3. Answer questions within 10 seconds
4. Get your final score
5. View leaderboard
